## Notes

- Keep the avatar and handle identical across all three. Cross-platform
  recognition is most of what a small brand's profile is doing.
- Don't overclaim. The Chrome Web Store certification already binds you to an
  accurate description of the product; contradicting it on social is a bad look
  and a genuine review risk.
- Pin an intro video on each profile once you have one — a 20-second "what it is
  and what it doesn't do" beats any bio.
